Pokédex: listado de Pokémon

Pokédex es una enciclopedia virtual que, partiendo de la API de Pokémon, nos permite conocer los datos básicos de los Pokémon: su nombre, número de Pokémon y tipo/s (cada Pokémon puede tener hasta dos diferentes). La información se completa con una imagen para que podamos conocer su aspecto.
